🚨 TROUBLESHOOTING
Problem: npm install fails
└─ Solution: npm cache clean --force && npm install

Problem: Port 5173 in use
└─ Solution: npm run dev -- --port 3000

Problem: API not connecting
└─ Solution: Ensure backend runs on https://localhost:7222

Problem: CORS error
└─ Solution: Configure CORS in Program.cs (backend)

Problem: Styles not loading
└─ Solution: Restart dev server and clear browser cache

Problem: Redirect to login
└─ Solution: Ensure user is stored in localStorage by login page
